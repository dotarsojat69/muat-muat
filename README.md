## Installation

1. Clone this repository to your local machine using:

   ```bash
   git clone https://github.com/dotarsojat69/muat-muat.git
   ```

2. Navigate to the project directory:

   ```bash
   cd muat-muat
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

## Usage

1. Start the development server:
   ```bash
   npm run dev
   ```
